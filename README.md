# Description
This Nasm x64 Assembly program lets you en- and decode everything passed to the stdin.

## Up and running
1. Clone this repository and cd into it
2. run `make`
3. `./encode < myFile.txt`
4. `./decode < myEncodedFile.txt`

## Testing
Running the test suite: `./test-suite.sh`.
